# Gem Crush Game

Match gems and solve puzzles in this colorful and addictive match-3 game.

## Features
- 500+ levels
- Special power-ups
- Daily challenges
- Events and rewards
- Beautiful gem graphics

## Build
```bash
npm install
npx cap sync
npx cap open android
npx cap open ios
```

## Tech Stack
- React + Vite
- Capacitor
- iOS & Android
